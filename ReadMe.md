# Expense Tracker (Python CLI)

A simple command-line expense tracker built with Python. It logs expenses to a CSV file, shows summaries, and visualizes spending with a pie chart.

## Features
- Add expenses with category and amount.
- View total spending and category-wise summary.
- Visualize expenses with a pie chart using `matplotlib`.
